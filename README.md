# How To Use
## Set Up
- Insert a ScreenGUI inside StarterGUI
- Insert a TextButton inside the ScreenGuI
- Name The TextButton Button
- Insert a Frame inside the ScreenGUI
- Insert a Textbutton inside the Frame
- Name the TexButton Teleport
- Insert a Local Script inside Button and Teleport

- It Should Looked like this:

Note: I changed the background color for Teleport for demonstration

- If everything is correct inside the property tab change set the frame visible to false
  <img width="294" height="760" alt="image" src="https://github.com/user-attachments/assets/261ee718-96e9-4f51-9ca8-dfe2d293d45c" />

- Inside Workspace, Insert a Part name it Point
- Raise the Part a little bit
- In the Property tab make sure CanCollide is false

## Code Set Up
- Copy the Content of TeleportCode to the Local script inside Teleport
- Copy the Content of ButtonCode to the Local script inside Button
